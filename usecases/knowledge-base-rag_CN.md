# 个人知识库 (RAG)

你整天阅读文章、推文和观看视频，但永远找不到上周看到的那个东西。书签堆积起来并变得无用。

这个工作流程从你保存的所有内容构建可搜索的知识库：

• 将任何 URL 放入 Telegram 或 Slack，它会自动摄取内容（文章、推文、YouTube 转录、PDF）
• 对你保存的所有内容进行语义搜索："我保存了关于智能体记忆的什么？"返回带有来源的排名结果
• 输送到其他工作流程 —— 例如，视频创意流水线在构建研究卡片时查询相关保存内容的 KB

## 所需技能

- [knowledge-base](https://clawhub.ai) 技能（或使用嵌入构建自定义 RAG）
- `web_fetch`（内置）
- 用于摄取的 Telegram 主题或 Slack 频道

## 如何设置

1. 从 ClawdHub 安装 knowledge-base 技能。

2. 创建一个名为 "knowledge-base" 的 Telegram 主题（或使用 Slack 频道）。

3. 提示 OpenClaw：
```text
当我在 "knowledge-base" 主题中放入一个 URL 时：
1. 获取内容（文章、推文、YouTube 转录、PDF）
2. 摄取到知识库并包含元数据（标题、URL、日期、类型）
3. 回复确认：摄取了什么和块数

当我在这个主题中问问题时：
1. 语义搜索知识库
2. 返回带有来源和相关摘录的顶级结果
3. 如果没有好的匹配，告诉我

此外：当其他工作流程需要研究时（例如，视频创意、会议准备），自动查询知识库以获取相关保存的内容。
```

4. 通过放入几个 URL 并询问诸如"我有什么关于 LLM 记忆的内容？"之类的问题来测试它。
