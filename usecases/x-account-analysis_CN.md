# X 账号分析

有许多网站旨在为您提供 X 账号的定性分析。虽然 X 已经为您提供了**分析**部分，但它更侧重于展示您的表现数据。

但定性分析专注于您的帖子质量，而不是表现统计数据。您可以从这种类型的分析中获得一些见解：
- 让我的帖子爆火的模式是什么？
- 我谈论哪些主题能获得最多的互动？
- 为什么我会得到 1000+ 点赞的帖子，但有时帖子点赞数 <5？我做错了什么？

有许多网站和应用程序旨在为您提供 X 分析，但它们专注于统计数据。可能只有 1-2 个网站允许您与 AI 对话来了解您的表现。

但现在您可以使用 OpenClaw 为您进行这种分析，而无需在这些网站上支付 10-50 美元的订阅费。

## 所需技能
Bird 技能。`clawhub install bird`（它已预装）

## 如何设置

以下是流程：
1. 确保 Bird 技能正常工作。
2. 出于安全和隔离考虑，您最好为 ClawdBot 创建一个新账户。
3. 使用您的 X 账户进行身份验证。登录 x.com（Chrome/Brave），并提供正确的 cookie 信息（`auth-token`、`ct0`），以便 OpenClaw 可以访问您的账户。
4. 要求 OpenClaw 查看您的真实账户，获取最近 N 条推文，并向它询问任何您喜欢的问题。或者，您可以要求它为您编写特定的脚本。
