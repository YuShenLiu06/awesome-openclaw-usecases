# 第二大脑

你有了想法、找到有趣的链接、听说要读的书 —— 但你没有一个好的系统来捕获它们。Notion 变得复杂，Apple Notes 成为 10,000 个未读条目的墓地。你需要像给朋友发短信一样简单的东西。

这个工作流程将 OpenClaw 转变为一个你通过短信交互的记忆捕获系统，由一个你可以随时浏览的自定义可搜索 UI 支持。

## 功能说明

- 通过 Telegram、iMessage 或 Discord 向你的 OpenClaw 发短信任何内容 —— "提醒我读一本关于本地 LLM 的书" —— 它会立即记住
- OpenClaw 的内置记忆系统永久存储你告诉它的一切
- 一个自定义 Next.js 仪表板让你搜索每个记忆、对话和笔记
- 全局搜索（Cmd+K）跨所有记忆、文档和任务
- 没有文件夹、没有标签、没有复杂的组织 —— 只有文本和搜索

## 痛点

每个笔记应用程序最终都会变成一件苦差事。你停止使用它，因为组织的摩擦比忘记的摩擦更大。关键见解是：**捕获应该像发短信一样容易，检索应该像搜索一样容易**。

## 所需技能

- Telegram、iMessage 或 Discord 集成（用于基于短信的捕获）
- Next.js（OpenClaw 为你构建它 —— 无需编码）

## 如何设置

1. 确保你的 OpenClaw 连接到你首选的消息平台（Telegram、Discord 等）。

2. 立即开始使用它 —— 只需给你的机器人发短信你想要记住的任何内容：
```text
嘿，提醒我读"设计数据密集型应用程序"
保存此链接：https://example.com/interesting-article
记住：John 推荐了 5th 街道的餐厅
```

3. 通过提示 OpenClaw 构建可搜索 UI：
```text
我希望你构建一个第二大脑系统，让我可以审查我们所有的笔记、
对话和记忆。请用 Next.js 为我构建它。

包括：
- 所有记忆和对话的可搜索列表
- 跨所有内容的全局搜索（Cmd+K）
- 按日期和类型过滤的能力
- 干净、最简化 UI
```

4. OpenClaw 将为你构建和部署整个 Next.js 应用程序。导航到它提供的 URL，你就有了你的第二大脑仪表板。

5. 从现在开始，每当你想到什么时 —— 在路上、在会议中、睡前 —— 只需给你的机器人发短信。需要查找任何内容时，回到仪表板。

## 核心见解

- 力量在于**零摩擦捕获**。你不需要打开应用程序、选择文件夹或添加标签。只需发短信。
- OpenClaw 的记忆系统是累积的 —— 它记住你曾经告诉它的*所有内容*，随着时间的推移使其更强大。
- 你可以从手机给你的机器人发短信，它在你的计算机上构建东西。界面就是对话。
