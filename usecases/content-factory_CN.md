# 多智能体内容工厂

你是一个在多个平台之间从事研究、写作和设计的内容创作者。每一步 —— 查找热门主题、编写脚本、生成缩略图 —— 占据你一天中的数小时。如果一个专业智能体团队在隔夜处理所有内容会怎么样？

这个工作流程在 Discord 内部设置一个多智能体内容工厂，其中不同的智能体在专用频道中处理研究、写作和视觉资产。

## 功能说明

- **研究智能体** 每天早上扫描热门故事、竞争对手内容和社交媒体以获取最佳内容机会
- **写作智能体** 接收顶级想法并编写完整的脚本、线程或新闻通讯草稿
- **缩略图智能体** 为内容生成 AI 缩略图或封面图像
- 每个智能体在自己的 Discord 频道中工作，保持所有内容有组织且可审查
- 按计划自动运行（例如，每天上午 8 点），这样你醒来时面对完成的内容

## 痛点

内容创作有三个阶段 —— 研究、写作和设计 —— 大多数创作者都在手动处理所有三个阶段。即使有 AI 写作工具，你仍然必须一次提示它们一个。这个系统将智能体链接在一起在一个流水线中，其中一个智能体的输出提供给下一个，完全无需动手。

## 所需技能

- 带有多个频道的 Discord 集成
- `sessions_spawn` / `sessions_send` 用于多智能体编排
- [x-research-v2](https://clawhub.ai) 或类似用于社交媒体研究
- 本地图像生成（例如，Nano Banana）或图像生成 API
- [knowledge-base](https://clawhub.ai) 技能（可选，用于 RAG 驱动的研究）

## 如何设置

1. 设置一个 Discord 服务器（或让 OpenClaw 为你做 —— 只需说"为我们设置一个 Discord"）。

2. 为每个智能体创建频道：
   - `#research` —— 热门主题和内容机会
   - `#scripts` —— 编写的草稿和大纲
   - `#thumbnails` —— 生成的图像和封面艺术

3. 提示 OpenClaw：
```text
我希望你在 Discord 内部为我构建一个内容工厂。
为不同的智能体设置频道：

1. 研究智能体（#research）：每天早上 8 点，研究顶级热门
   故事、竞争对手内容以及社交媒体上在我的细分市场中
   表现良好的内容。发布带有来源的前 5 个内容机会。

2. 写作智能体（#scripts）：接收研究智能体的最佳想法
   并编写完整的脚本/线程/新闻通讯草稿。将其发布在 #scripts 中。

3. 缩略图智能体（#thumbnails）：为内容生成 AI 缩略图或封面图像。
   将它们发布在 #thumbnails 中。

让他们所有的工作在不同的频道中有组织。
每天早上自动运行此流水线。
```

4. 针对你的平台进行自定义：
```text
我专注于 X/Twitter 线程，而不是 YouTube。更改写作智能体
以生成推文线程而不是视频脚本。
```

## 核心见解

- 力量在于**链接的智能体** —— 研究提供写作，写作提供缩略图。你不需要单独提示每个步骤。
- Discord 频道使得很容易单独审查每个智能体的工作并给出反馈，如"脚本太长"或"更专注于 AI 新闻。"
- 你可以将其调整为任何内容格式：推文、新闻通讯、LinkedIn 帖子、播客大纲、博客文章。
- 运行本地模型进行图像生成（如 Mac Studio 上的 Nano Banana）可以降低成本并给你更多控制。
